# My Portfolio

A simple, responsive photography and videography portfolio built with HTML and CSS.

## Features

✨ **Responsive Design** - Looks great on desktop, tablet, and mobile
📸 **Photo Gallery** - Showcase your photography
🎬 **Video Gallery** - Embed YouTube videos
📱 **Mobile Friendly** - Optimized for all devices
🎨 **Modern Styling** - Clean and professional design

## Getting Started

### 1. Enable GitHub Pages

1. Go to your repository settings
2. Scroll to "GitHub Pages"
3. Under "Source", select the **main** branch
4. Your site will be available at: `https://laxulilies.github.io/laxulilies/`

### 2. Customize Your Portfolio

Edit the following files:

#### `index.html` - Main Content
- Change "My Portfolio" to your name or portfolio title
- Update the hero section text
- Replace placeholder images with your own
- Add your social media links
- Update the about section with your bio
- Add your email and social media links

#### `styles.css` - Design
- Change colors in the CSS variables
- Adjust fonts, spacing, and layout
- Customize the gradient colors

## How to Add Your Content

### Add Photos
1. Replace the placeholder image URLs in the gallery section
2. Update photo titles in the overlay text

```html
<div class="gallery-item">
    <img src="your-image-url.jpg" alt="Your Photo">
    <div class="overlay">
        <p>Your Photo Title</p>
    </div>
</div>
```

### Add Videos
Replace the YouTube embed URLs with your own video links:

```html
<div class="video-item">
    <iframe width="100%" height="300" src="https://www.youtube.com/embed/YOUR_VIDEO_ID" frameborder="0" allowfullscreen></iframe>
</div>
```

## Image Hosting Options

- **GitHub**: Upload images to your repo (create an `images/` folder)
- **Imgur**: Free image hosting - get direct links
- **Cloudinary**: Free tier for media files
- **Your own server**: If you have hosting

## Tips

💡 Keep file sizes small for faster loading
💡 Use high-quality images (but optimized)
💡 Update your contact information
💡 Test on mobile devices before sharing
💡 Add more sections as needed (testimonials, services, etc.)
